# 🎬 Movie Recommendation System

## 📄 Research Paper Reference
**Title:** Book Recommendation System Using Machine Learning  
**Source:** ResearchGate  

##  Project Overview
This project implements a **Movie Recommendation System** as a web-based interface,
inspired by the design and system architecture described in the research paper.
The system allows users to search for movies and get visual recommendations
displayed in a clean, Netflix-style dark themed interface.

## 🎨 Design Features
- 🌑 Dark themed UI with overlay background
- 🎥 Netflix-style horizontal scrollable movie cards
- 🖱️ Hover zoom effect on movie posters
- 📱 Clean and minimal layout
- 🔴 Red accent buttons (search/action buttons)
